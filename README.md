# Attendance ERP System

Ứng dụng chấm công tích hợp với ERP qua JSON-RPC

## Setup

1. Clone project
2. Copy `.env.example` thành `.env.local`
3. Điền thông tin ERP vào `.env.local`
4. Cài dependencies: `npm install`
5. Chạy dev: `npm run dev`
6. Deploy: Push lên GitHub và import vào Vercel

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Lucide React Icons
